FinTrack — Firebase Setup

One-time setup to enable multi-device sync

1Create a Firebase Project

Go to console.firebase.google.com → Click "Add project" → Give it a name (e.g. fintrack) → Continue until project is created.

2Enable Google Sign-In

In your project → Build → Authentication → Get started → Sign-in method tab → Under "Additional providers" click Google → Toggle Enable → Set a support email → Save.

3Create Firestore Database

In your project → Firestore Database → Create database → Choose Production mode → Select a region → Done.

Then go to Rules tab and replace the rules with:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }

Click Publish.
